## 🚀 Overview

This project is a **Call Title AI Agent** that generates meaningful titles based on user conversations.

---

## 🧠 How it works

- Uses embeddings (SentenceTransformer)
- Stores data using FAISS
- Backend built with FastAPI
- Frontend using Angular

---

## ⚙️ Features

- Generate smart call titles
- Context-aware responses
- Fast retrieval using vector search
